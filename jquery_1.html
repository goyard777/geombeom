제이쿼리는 무엇인가?

제이쿼리는 자바스크립트와의 차이점이 있다 제이쿼리는 동적인 웹페이지를 브라우저 종류에 상관없이 동일한 방식으로 구현할수있다 이는 자바스크립트에서 구현하기 어렵거나 불편했던 기능을 손쉽게 구현할수 있도록 지원하며 이벤트처리 애니메이션 dom관리 ajax응용 같은 프로그래밍 작업을 자바스크립트 보다 좀더 간단하게 할수 있다 제이쿼리의 큰장점은 css html javascript 3가지 모두 관리할수있으며 제작할수있다.

제이쿼리는 자바스크립트의 명령문을 생락할수있으며 파일을 따로 제작가능하며 지원가능하다는점에 자바스크립트 보다 손쉽게 사용가능하다 그렇다면 자바스크립트는 왜사용하는것일까? 라는 의문점이 생길것이다

제이쿼리는 어디까지나 이 3가지를 관리 제작을 할수있다는점 이고 정작 자바스크립트의 태그를 이용해야하는 점은 변함없다 이는 자바스크립트이 개념이 잡혀있지않으면 제이쿼리의 제작에 어려움 있고 이는 자바스크립트의 응용이라고 생각하여야 한다 잊지 말자 제이쿼리는 어디까지나 범용성 이다
