이미지 관련 태그

이미지는 height width 두속성을 모두 생략하면 이미지의 원본 크기로 표시되고 그렇지 않으면 사이즈를 지정할수있다 이미지 생성시 alt 라는 속성을 적을수있는데 (생략가능) 이는 오류나 전송 오류로 표시하지 못할경우 대체할수있는 문자열을 제공한다. 이미지는 jpg,gif png 사용가능하다 이미지에서 불러올 주소를 입력하는것은 src 구분된다

오디오 태그 및 비디오 태그

오디오나 비디오를 재생하기위해 사용되는 태그이며 오디오는 audio 비디오는 video 태그로 웹문서에서 실행하게 해준다 오디오 태그는 이미지 태그와 비슷하게 src 주소속성을 이용하며 파일 경로를 지정한다 오디오 태그의 특징은 제어기능을 입력할수 있으며 자동플레이 연속플레이등 값을 지정할수 있다

비디오 태그는 source 태그를 사용하여 src주소로 입력해줘야한다 이유는 source 특징은 재생 속성을 상관없이 동일하게 재생하게 해주는 역활을한다 비디오 태그도 오디오 태그와 동일하게 자동재생 연속재생등 여러 컨트롤러를 재생한다 아래 코드를 참조해보자
