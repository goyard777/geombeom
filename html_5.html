공간 분할 태그

공간 분할 태그는 태그 자체의 특별한 의미나 기능이 없다. 단지 페이지 공간 분할의 단위로 사용되며 같은 공간에 위치를 엘리먼트 역활을ㄹ하며 특정공간 배치를 쉽게 하도록 한다 div, span 코드에서는 글자를 나타냈지만 그림 사진 동영상등 웹문서의 공간을 분할 할수 있도록 도와준다 div는 블록 형식 공간 분할 span은 인라인 형식 공간 분할된다.

시멘틱 태그

시멘틱 태그는 웹페이지 구성 방식의 변화도를 나타내며 웹페잊 구조 div span등 태그를 이용하여 페이지 공간을 적절히 분할 시켜주며 물리적 영역을 구별하므로써 효과으로 페이지 공간 구별 할수있다 -시멘틱 태그- header 머리말 nav 메뉴 영역 aisde 보조영역 section 형식적 구분영역 aricle 내용적 구분 footer 꼬리말 영역 이 모든 시멘틱태그와 div span 태그를 이용한 모습을 보고 참고하길바란다.
