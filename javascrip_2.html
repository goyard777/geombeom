href의 자바스크립트 코드 작성 한다면?

html 에서 배운 href 의 개념은 확실히 잡혀있는가? 어떤 기능을 하는지 알고있다면 자바스크립트의 코드를 넣어 작동시키는것을 구현해본다 예시를 보고 이해 해보자

자바 스크립트의 alert 는 문자를 입력해주는 html 에서 h 태그와 같은 기능과 비슷하지만 스크립트에서는 단지 출력만 해주는 기능이다 .

자바스크립트로 html 콘텐츠 출력

자바스크립트 코드로 html 콘텐츠 웹 페이지에 직접 삽입하여 바로 브라우저 윈도우에 출력되게 할수 있다

document.writeln() 사용하면된다 주로 사용되니 기억해두자

여기서 글자 출력이 필요한 이유는 1장에서 말했다시피 html에서는 그저 출력만 했다면 여기서부턴 동적인 일을 하는것을 출력하기때문에 따로 태그를 달아주는것이다 이는 그저 출력문이 아니라는걸 명심해야한다.!
